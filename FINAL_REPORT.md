# RetailFlow Capstone Project

## Overview

RetailFlow is an end-to-end retail analytics pipeline built using AWS services.

## Technologies Used

- Python
- Pandas
- Faker
- AWS S3
- AWS Glue
- Glue Data Catalog
- Amazon Athena
- SQL
- Git
- VS Code

## Pipeline

Synthetic Data
→ Amazon S3 (Raw)
→ AWS Glue ETL
→ Curated Parquet
→ Glue Catalog
→ Athena SQL Analytics

## Analytics

Implemented:

- Daily Sales
- Customer Summary
- Clickstream Analysis
- Athena Views

## Data Engineering Features

- ETL using PySpark
- Schema Evolution
- Data Quality Rules
- Glue Job Bookmarks
- Documentation

## Future Enhancements

- AWS Lake Formation Governance
